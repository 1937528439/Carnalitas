Sex Scene System
------------------------

General-purpose scripted effect to show a sex scene event:
* carn_sex_scene_effect (requires PLAYER - character, TARGET - character)

New on_action to add sex scene events to:
* carn_sex_scene

Scripted effect to request flags for sex scenes:
* carn_sex_scene_request_[FLAG]

In the trigger block of sex scene events, use this "trigger" (actually an effect) to set flags:
* carn_sex_scene_is_[FLAG]

After setting flags, use this scripted trigger to check against requested flags:
* carn_sex_scene_matches_requested_flags_trigger

An example of using carn_sex_scene_effect to trigger a dominant rape scene:

 effect = {
 	carn_sex_scene_request_dom_player = yes
 	carn_sex_scene_request_vaginal = yes
 	carn_sex_scene_request_noncon = yes
 	carn_sex_scene_effect = {
 		PLAYER = root
		TARGET = scope:target
 	}
 }

List of supported flags:

dom_player
sub_player
oral
vaginal
anal
handjob
masturbation
consensual
noncon
dubcon
painful
bdsm
bondage
toy
watersports
scat
gore

There are game rules to turn off anal, noncon, dubcon, painful, watersports, scat, and gore.

An example sex scene can be found in common/events/sex_scenes/carn_example_sex_scene_events.txt
