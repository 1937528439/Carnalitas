Features
------------------------

* Game rule to remove all STDs

* New congenital trait group: Small Dicks (dick_small_1 to dick_small_3)
* New congenital trait group: Big Dicks (dick_big_1 to dick_big_3)
* New congenital trait group: Small Tits (tits_small_1 to tits_small_3)
* New congenital trait group: Big Tits (tits_big_1 to tits_big_3)

* New traits: futa, futa_genetic (names differ depending on game rule)
Futas currently can't father children or marry other women.

------------------------
Sex Scene System
------------------------

General-purpose scripted effect to show a sex scene event:
* carn_sex_scene_effect (requires PLAYER - character, TARGET - character)

New on_action to add sex scene events to:
* carn_sex_scene

Scripted effect to set flags for sex scenes:
* carn_sex_scene_set_[FLAG]

Scripted trigger to check flags for sex scenes:
* carn_sex_scene_is_[FLAG]

An example of using carn_sex_scene_effect to trigger a dominant rape scene:

 effect = {
 	carn_sex_scene_set_dom_player = yes
 	carn_sex_scene_set_vaginal = yes
 	carn_sex_scene_set_noncon = yes
 	carn_sex_scene_effect = {
 		PLAYER = root
		TARGET = scope:target
 	}
 }
